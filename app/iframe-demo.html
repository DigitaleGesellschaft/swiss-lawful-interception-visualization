How to Embed the Visualisation

To embed the visualisation into your own website, you can use an iframe. The site is optimized for a mininum width of 680px.
The base url is https://www.digitale-gesellschaft.ch/sliv. You can directly open the site in a preset filter state by defining a language, year and canton: 

			de
			de/2015
			de/2015/zh

Where the two letter language prefix is optional, and defaults to de. Other languages: rm, fr, en, it. For example if you want to directly load the german version of Zurich in 2015 you can use the following path: 

https://www.digitale-gesellschaft.ch/sliv/de/2015/zh
